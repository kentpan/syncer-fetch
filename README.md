# syncer-fetch
一个适合下载上传二进制内容的fetch封装库，支持代理、并发控制、重试机制等功能。

## 特性

- 🚀 基于原生fetch API的轻量级封装
- 🔄 支持请求重试和超时控制
- 🌐 内置代理支持，可配置多个代理自动切换
- 📦 支持并发控制，防止过多请求导致性能问题
- 📊 自动处理不同类型的响应数据（图片、文本、二进制等）
- 📁 支持文件上传和下载进度监控
- 🔧 支持请求前后钩子函数
- 📝 完善的错误处理和状态码映射
- 🗜️ 支持Base64编码/解码
- 📦 支持多种模块格式（UMD、CommonJS、ES Module）

## 安装

```bash
npm install syncer-fetch
